# Estrutura de Arquivos gerados

Após seguir o tutorial sua estrutura de arquivos deve ficar da seguinte maneira:

```
svelte-error-screen
├── node_modules
├── package.json
├── rollup.config.js
├── yarn.lock
├── public
│   ├── build
│   ├── index.html
└── src
    ├── dev
    │   ├── App.svelte
    │   └── main.js
    └──  lib
        └── index.svelte
```
